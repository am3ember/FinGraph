# FinGraph 全球经济仪表盘 — 指标体系设计

## 设计原则

1. **信号而非数据**：不追踪原始数值（利率、股价），而是追踪"方向+强度+预期偏离"
2. **可聚合**：每条图文产生一个打分，多条打分聚合为板块指数
3. **时间衰减**：近期数据权重更高，远期逐渐衰减
4. **可追溯**：每个指数的变动可以溯源到具体的图文条目

## 打分逻辑

每条图文数据 → 标签分类 + 情绪打分：
- **+2**: 强正面（远超预期、创历史新高、大幅反弹）
- **+1**: 温和正面（好于预期、改善、扩张）
- **0**: 中性（持平、符合预期）
- **-1**: 温和负面（不及预期、放缓、收窄）
- **-2**: 强负面（暴跌、创新低、远差于预期、崩盘）

## 指数计算方法

### 单日打分
```
每篇文章 → 每条图文 → (区域标签, 指数标签, 情绪分值 -2~+2)
```

### 指数聚合（7日滚动加权均值）
```
Index(t) = Σ(score_i × weight_i) / Σ(weight_i)

weight_i = decay^(days_ago)  // 时间衰减，decay=0.85
```

### 标准化输出
```
最终指数 = 50 + Index(t) × 25  // 映射到 0-100 区间
- 80-100: 极度乐观/过热
- 60-80:  乐观
- 40-60:  中性
- 20-40:  悲观
- 0-20:   极度悲观/恐慌
```

## 指数更新示例

以"中国房地产指数"为例：

| 日期 | 事件 | 分值 |
|------|------|------|
| 5月20日 | "多省下调首付比例至15%"（政策利好） | +1 |
| 5月21日 | "4月30城新房销售面积同比下降23%"（数据恶化） | -2 |
| 5月22日 | "一线城市二手房挂牌量创历史新高"（供给过剩） | -1 |

7日滚动指数 = 50 + weighted_avg(-0.67) × 25 ≈ 33（悲观区间）

## 实现路径

1. **标签分类器**：将每条图文打上 (区域, 指数) 标签
2. **情绪打分器**：对每条图文评估 -2~+2 的情绪分值
3. **指数引擎**：按时间衰减聚合，输出每日指数值
4. **存储**：每日指数值写入时序数据库
5. **API**：供前端仪表盘调用

标签分类和情绪打分推荐用 LLM（准确度高），辅以关键词规则作为 fallback。
