## 各区域详细说明

### 1. 左侧指数面板（固定宽度 280px）

垂直排列 6 个指数卡片，每个卡片包含：
- 指数名称
- 当前数值（大字）
- 进度条/色带（颜色随分值变化：红<30, 橙30-45, 灰45-55, 浅绿55-70, 绿>70）
- 较昨日变化箭头（↑ / ↓ / →）
- 点击某个指数 → 底部趋势图切换到该指数 + 右侧信号流筛选为该指数

### 2. 中央地图区域（自适应宽度）

- 中国省级行政区划地图（2D，暗色底）
- 脉冲动画：最近 30 天内文章提及的省份/城市，显示信号脉冲
  - 绿色涟漪 = 正面信号（score > 0）
  - 红色涟漪 = 负面信号（score < 0）
  - 涟漪大小 = |score|（±2 大涟漪，±1 小涟漪）
  - 多条信号叠加时脉冲频率加快
- 悬浮 tooltip：鼠标悬停在脉冲上 → 显示"深圳: 写字楼空置率飙升29% [-2]"
- 点击脉冲 → 右侧详情面板展开该条完整数据

### 3. 脉冲详情弹出卡片（点击触发）

点击地图上的脉冲点，在该位置附近弹出浮动卡片：
- 城市/省份名称 + 日期
- 数据摘要（完整文字）
- 命中指数 + 分值 + 信号类型
- 对应图表图片（如有）
- [查看原文] 链接
- 点击卡片外区域或 × 按钮关闭

同一位置有多条信号时，卡片内以列表形式展示，可上下滚动

### 4. 底部趋势区域（固定高度 200px）

- 默认显示所有 6 个指数的 90 天趋势折线（叠加在一张图上，各线颜色不同）
- Tab 切换：可选择只看某一个指数的单独折线（更大更清晰）
- X 轴：日期
- Y 轴：0-100
- 辅助线：50分中性线（虚线）
- 标注：历史均值区间（浅灰色带）
- 交互：悬停某天 → tooltip 显示当天6个指数的具体数值

## 交互逻辑

| 操作 | 响应 |
|------|------|
| 点击左侧某个指数卡片 | 底部趋势图高亮该指数；地图脉冲只显示该指数相关的信号 |
| 点击地图上的脉冲 | 弹出该位置的详情卡片（数据摘要+分值+图表） |
| 悬停地图脉冲 | 显示简短 tooltip（城市+摘要+分值） |
| 点击底部趋势图某天 | 地图上高亮该天产生的所有脉冲 |
| 点击顶部导航其他区域 | 切换到对应区域页面（布局相同，指数/地图不同） |
| 再次点击已选中的指数卡片 | 取消筛选，恢复显示全部脉冲 |

## 响应式适配

- 1920px+：完整左栏 + 地图布局
- 1440px：左侧面板缩窄
- 1024px（平板）：指数面板移至地图上方横向排列
- 768px（手机）：纯竖向布局 → 指数卡片横向滚动 + 地图全宽 + 趋势图全宽
